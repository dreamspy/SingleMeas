  Thorlabs TXP PAX VISA Instrument Driver

      This VISA instrument driver module provides programming support for the
      Thorlabs TXP PAX Polarization Analyzer module.
      
________________________________________________________________________________

              The following functions are in alphabetical order.

________________________________________________________________________________

  TXPPAX_close

      ViStatus TXPPAX_close (ViSession instrumentHandle);

  Purpose

      This function performs the following operations:
      viClose (instrSession) and viClose (rmSession).
      
      Note:
      -----
      The instrument must be reinitialized to use it again.

  Parameter List

      instrumentHandle

          Variable Type       ViSession

          This parameter accepts the Instrument Handle returned by the
          Initialize function to select the desired instrument driver session.
          

  Return Value

          This value shows the status code returned by the function call.
          
          For status codes see function <Error Message>.
          
________________________________________________________________________________

  TXPPAX_ConvertLimits

      ViStatus TXPPAX_ConvertLimits (ViSession instrumentHandle,
                                     ViReal64 dispMin, ViReal64 dispMax,
                                     ViReal64 dispDefault, ViPReal32 dispMin,
                                     ViPReal32 dispMax,
                                     ViPReal32 dispDefault);

  Purpose

      This function converts ViReal64 (double precision) adjustment limit
      values to ViReal32 (single precision) limit values.
      
      Note: This function is for service purposes only.
      
      Required access level: None
      

  Parameter List

      instrumentHandle

          Variable Type       ViSession

          This parameter accepts the Instrument Handle returned by the
          Initialize function to select the desired instrument driver session.
          
          Default value: VI_NULL
          

      dispMin

          Variable Type       ViReal64

          This parameter specifies the minimum display value.
          

      dispMax

          Variable Type       ViReal64

          This parameter specifies the maximum display value.
          

      dispDefault

          Variable Type       ViReal64

          This parameter specifies the default display value.
          

      dispMin

          Variable Type       ViReal32 (passed by reference)

          This parameter returns the minimum display value.
          

      dispMax

          Variable Type       ViReal32 (passed by reference)

          This parameter returns the maximum display value.
          

      dispDefault

          Variable Type       ViReal32 (passed by reference)

          This parameter returns the default display value.
          

  Return Value

          This value shows the status code returned by the function call.
          
          This function allways returns VI_SUCCESS.

  TXPPAX_GetAccessLevel

      ViStatus TXPPAX_GetAccessLevel (ViSession instrumentHandle,
                                      ViPUInt32 effectiveAccessLevel,
                                      ViPUInt32 maximumAccessLevel);

  Purpose

      This function queries the user's effective access level and the user's
      maximum access level.
      
      The values used by this function are 32bit bitmaps which are defined in
      the driver's header file. A set bit in the according value indicates that
      the access is granted.
      
      Description:
      
      Value      Definition                Description
      ---------------------------------------------------------------
      0x00000001 TXPPAX_ACCKEY_SERNR       Reserved
      0x00000002 TXPPAX_ACCKEY_BLD_DATE    Reserved
      0x00000004 TXPPAX_ACCKEY_REP_DATE    Reserved
      0x00000008 TXPPAX_ACCKEY_ADJ_DATE    Reserved
      0x00000010 TXPPAX_ACCKEY_CAL_DATE    Reserved
      0x00000020 TXPPAX_ACCKEY_USER_TXT    Set card's user text 0x00000100
      TXPPAX_ACCKEY_SERVICE     Reserved
      0x00000200 TXPPAX_ACCKEY_STATE       Set state/operating mode 0x00000400
      TXPPAX_ACCKEY_SETVAL      Set configurationo values
      
      Other values: Reserved
      
      Required access level to execute this function: None
      

  Parameter List

      instrumentHandle

          Variable Type       ViSession

          This parameter accepts the Instrument Handle returned by the
          Initialize function to select the desired instrument driver session.
          

      effectiveAccessLevel

          Variable Type       ViUInt32 (passed by reference)

          This parameter returns the user's effective access level. Command
          execution is checked against the effective access level.
          
          Note: You may pass VI_NULL for this parameter.
          

      maximumAccessLevel

          Variable Type       ViUInt32 (passed by reference)

          This parameter returns the user's maximum access level. The maximum
          access level is specified by the user's access key stored in the
          system registry.
          
          Note: You may pass VI_NULL for this parameter.
          

  Return Value

          This value shows the status code returned by the function call.
          
          For status codes see function <Error Message>.

  TXPPAX_GetAdcAdjObjectFP32FP32

      ViStatus TXPPAX_GetAdcAdjObjectFP32FP32 (ViSession instrumentHandle,
                                               ViUInt8 type,
                                               ViUInt32 limitsCommand,
                                               ViUInt32 pointsCommand,
                                               ViPUInt8 validity,
                                               ViPReal32 dispMin,
                                               ViPReal32 dispMax,
                                               ViReal32 _VI_FAR xValues[],
                                               ViReal32 _VI_FAR yValues[],
                                               ViUInt32 size,
                                               ViPUInt32 returnCount);

  Purpose

      This function queries a card's ADC adjustment object (FP32-FP32).
      
      Note: This function will return VI_ERROR_TXP_MOD_CHECKSUM (0xBFFC0A09) as
      status code if the adjustment object was never written.
      
      Note: This function is for service purposes only.
      
      Required access level: TXPPAX_ACCKEY_ADJUST
      

  Parameter List

      instrumentHandle

          Variable Type       ViSession

          This parameter accepts the Instrument Handle returned by the
          Initialize function to select the desired instrument driver session.
          

      type

          Variable Type       ViUInt8

          This parameter specifies the type of the adjustment object.
          
          Valid values:
             TXPPAX_ADJOBJ_ADC_2P (2)
             TXPPAX_ADJOBJ_ADC_MP (3)
          
          Default value: TXPPAX_ADJOBJ_ADC_2P
          

      limitsCommand

          Variable Type       ViUInt32

          This parameter specifies the limits command number.
          
          Default value: 0
          

      pointsCommand

          Variable Type       ViUInt32

          This parameter specifies the points command number.
          
          Note: This value is ignored on TXPPAX_ADJOBJ_DAC_2P and
          TXPPAX_ADJOBJ_ADC_2P adjustment objects.
          
          Default value: 0
          

      validity

          Variable Type       ViUInt8 (passed by reference)

          This parameter returns the validity value.
          
          Note: You may pass VI_NULL.
          
          Valid values:
             TXPPAX_ADJ_INVALID    (0)

  TXPPAX_GetStatus

      ViStatus TXPPAX_GetStatus (ViSession instrumentHandle,
                                 ViPUInt32 eventRegister,
                                 ViPUInt32 statusRegister);

  Purpose

      This function queries the Event Register and the Status Register.
      
      The values used by this function are 32bit bitmaps which are defined in
      the driver's header file. A set bit in the Event Register indicates that
      the corresponding event has occured since the last call to this function.
      The bits in the Event Register are cleared when read. A set bit in the
      Status Register indicates that the corresponding state is currently
      active.
      
      Description:
      
      Value      Definition                    Description
      ---------------------------------------------------------------
      0x00000001 TXPPAX_STATBIT_CARD_EVENT     Card event
      0x00000002 TXPPAX_STATBIT_OVERTEMP       Overtemperature
      0x00000004 TXPPAX_STATBIT_COOLING        Cooling requirement 0x00000008
      TXPPAX_STATBIT_VCC_FAIL       Power supply error
      0x00000010 TXPPAX_STATBIT_ALARM          Chassis alarm
      0x00000020 TXPPAX_STATBIT_CARD_PRESENT   Card present (reserved)
      0x00000040 TXPPAX_STATBIT_CARD_READY     Card ready (reserved)
      0x00000080 TXPPAX_STATBIT_GPIO           General Purpose I/O 0x00000100
      TXPPAX_STATBIT_SENSOR_MISSING Sensor head missing
      0x00000200 TXPPAX_STATBIT_MACRO_ACTIVE   Macro active
      0x00000400 TXPPAX_STATBIT_TRIGGER_WAIT   Waiting for trigger
      0x00000800 TXPPAX_STATBIT_POWER_RANGE    Optical power too high
      0x00001000 TXPPAX_STATBIT_SENSOR_TEMP    Sens temperature error
      0x00002000 TXPPAX_STATBIT_SPEED          Sens speed out of range
      0x00004000 TXPPAX_STATBIT_SENSOR_ADJ     Sensor adjustment error
      0x00008000 TXPPAX_STATBIT_CARD_ADJ       Card adjustment error
      
      Other values: Reserved
      
      Required access level: None
      

  Parameter List

      instrumentHandle

          Variable Type       ViSession

          This parameter accepts the Instrument Handle returned by the
          Initialize function to select the desired instrument driver session.
          

      eventRegister

  TXPPAX_init

      ViStatus TXPPAX_init (ViRsrc resourceName, ViUInt32 slotNumber,
                            ViChar _VI_FAR userIdentification[],
                            ViUInt32 communicationTimeout,
                            ViUInt32 cardBootTimeout, ViUInt32 keepAliveTime,
                            ViPSession instrumentHandle);

  Purpose

      This function performs the following initialization actions:
      
      - Opens a session to the VISA Default Resource Manager and a
        session to the specified device using the interface and
        address specified in the <Resource Name> parameter.
      
      - Performs an identification query on the instrument.
      
      - Resets the instrument to a known state.
      
      - Reads Access Key data from the Registry if available and sends
        the the data to the instrument.
      
      - Sets the Access Level to the maximum allowed value.
      
      - Waits for the card to finish booting if specified.
      
      - Sets the data link Keep Alive Time if specified.
      
      - Returns an Instrument Handle which is used to differentiate
        between different sessions of this instrument driver.
      
      Note: Each time this function is invoked a unique Session is opened. To
      lock the resource use the VISA library <viLock> function.
      

  Parameter List

      resourceName

          Variable Type       ViRsrc

          This parameter specifies the VISA resource string of the device that
          is to be initialized.
          
          Based on the Resource Name, this operation establishes a
          communication session with a device. The syntax for the Resource Name
          is shown below. Optional parameters are shown in square brackets
          ([]).
          
          Syntax: "TCPIP[board]::<host address>::<port>::SOCKET"
          
          Default value: "TCPIP::localhost::2402::SOCKET"
          
          Note: The TXP default port number is 2402.
          
          Example Resource Strings:
          ----------------------------------------------------------------
          TCPIP::192.168.1.100::2402::SOCKET
          A device with the IP-address 192.168.1.100 and the standard port 2402
          is connected.
          
          TCPIP::localhost::2402::SOCKET
          A device connected to the users own machine using the standard port.
          
          TCPIP::dev.company.com::2544::SOCKET
          A device connected to the machine 'dev.company.com' using the IP-port
          2544 for the connection.
          

      slotNumber

          Variable Type       ViUInt32

          This parameter specifies the slot within the instrument chassis.
          
          Valid range:   0..63 (TXPPAX_MAX_SLOT_COUNT -1)
          Default value: 0
          
          Note: Slot numbers are zero based.
          

      userIdentification

          Variable Type       ViChar[]

          This parameter specifies a user identification string.
          
          Note:
          The string will be truncated to TXPPAX_ID_STRING_BUFFER_SIZE -1 (64)
          characters excluding the NUL character.
          
          You may pass VI_NULL to this parameter. In this case a user
          identification string will be generated automatically and will
          contain the user's machine hostname. For more information see also
          function <Get Module Information>.
          

      communicationTimeout

          Variable Type       ViUInt32

          This parameter specifies the TCP/IP communication timeout value in
          milliseconds [ms].
          
          Default value: 10000 ms
          
          Note: A timeout value of VI_TMO_INFINITE disables the timeout
          mechanism.
          

      cardBootTimeout

          Variable Type       ViUInt32

          This parameter specifies the maximum time in milliseconds [ms] the
          driver waits for the card to boot at initilization with this
          function.
          
          A value of 0 indicates that the driver should not wait but return
          immediately. In this case the function <Get Status> has to be called
          in a loop to determine that the card has finished booting.
          
          Default value: 15000ms
          

      keepAliveTime

          Variable Type       ViUInt32

          This parameter specifies the Keep Alive Time in milliseconds [ms].
          
          A value of 0 specifies infinite time so the keep alive functionality
          is disabled. A value greater than 0 disconnects the instrument if no
          communication takes place within the specified time.
          
          Default value: 0 (disabled)
          

      instrumentHandle

          Variable Type       ViSession (passed by reference)

          This parameter returns an Instrument Handle that is used in all
          subsequent function calls to differentiate between different sessions
          of this instrument driver.
          
          Note: Each time this function is invoked a unique Session is opened.
          It is possible to have more than one session open for the same
          resource. To lock/unlock the requested resource use the VISA library
          functions <viLock> and <viUnlock>.
          

  Return Value

          This value shows the status code returned by the function call.
          
          For Status Codes see function <Error Message>.

  TXPPAX_SetBasicMeasSpeed

      ViStatus TXPPAX_SetBasicMeasSpeed (ViSession instrumentHandle,
                                         ViReal64 basicMeasurementSpeed);

  Purpose

      This function sets the basic measurement speed.
      
      The basic measurement speed specifies the number of samples taken per
      second. Every half turn of the built-in waveplate one sample is taken.
      Therefore the basic measurement speed also specifies the rotation speed
      of the waveplate. This may be calculated using the formula:
      
        R(waveplate) [1/min] = Basic Measurement Speed [1/s] / 30
      
      Note: The basic measurement speed may be used for a 50/60 Hz rejection.
      
      Required access level: TXPPAX_ACCKEY_SETVAL
      
      

  Parameter List

          Variable Type       ViSession

          Initialize function to select the desired instrument driver session.

          This parameter specifies the basic measurement speed [1/s].
          
          Valid range: see function <Get Basic Measurement Speed Range>
          
          Default value: 66.7 /s.
          

  Return Value

          For status codes see function <Error Message>.
